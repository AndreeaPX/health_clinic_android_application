# Aplicatie pentru gestiunea statutului si a istoricului medical
Aplicația mobilă are la bază limbajul de programare Java, mediul principal de dezvoltare 
este Android Studio, fiind disponibilă pentru telefoane mobile cu sistem de operare Android, vizând minimul de API 21, corespunzător unui sistem Android 5.0 (Lollipop).
MSL ( abreviere pentru „My Simple Life”) a fost gândit ca brandul pentru un lanț de clinici private noi apărute în România. 
Aplicația mobilă de care dispune le promite utilizatorilor și celor care dețin sistemul în sine o modalitate simplă de administrare, 
stocare și centralizare a informațiilor cu caracter medical și implicit personal. Aplicația disponibilă pentru dispozitive cu 
sistem de operare Android le oferă pacienților posibilitatea de a realiza programări la clinicile, specializările ori doctorii 
pe care sistemul le deține, de a-și putea gestiona tratamentele și afecțiunile prin adăugarea acestora în aplicație,
de a oferi feedback constructiv interacțiunilor pe care le-au avut cu medicii și clinica pe care au vizitat-o și de a beneficia de
posibilitatea transformării imaginilor în format text, . În același timp, toate informațiile și datele cu caracter personal sunt păstrate 
în siguranță, fapt asigurat în special prin procesul de înregistrare în aplicație. Utilizatorul are controlul deplin în ceea ce privește adăugarea,
selectarea, actualizarea și ștergerea datelor pe care le posedă. Pe de altă parte, clinica ce deține baza de date poate urmării și stoca cu ușurință
informațiile amintite anterior, având de asemenea posibilitatea de a le modifica după propriul interes. 
Aceasta aplicatie are la baza un server realizat in php si baza de date MySQL, oferite the Apache.
